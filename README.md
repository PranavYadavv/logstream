# logstream

Real-time log aggregation and streaming pipeline

## Installation

```bash
npm install logstream
```

## Quick Start

```javascript
const logstream = require('logstream');
logstream.init();
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
